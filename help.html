Reminda Kanban for ArcGIS Online

Getting Started

Reminda uses your ArcGIS Online account to store boards and tasks. Any ArcGIS Online account can sign in to reminda.app — no separate registration is needed.

Creating a new board provisions a hosted Feature Service in your ArcGIS Online content, which requires a Creator (or higher) license. If your account is a Viewer or Editor, you can still open and work on boards that someone else has created and shared with you.

  1. Go to Sign In and authenticate with your ArcGIS Online account.
  2. If you have a Creator license, click New Board to create your first board.
  3. If a board has been shared with you, use Load Board on the board selection page to open it.

Boards

Each Reminda board is a hosted Feature Service in your ArcGIS Online account. Boards are created and loaded from the board selection page.

ActionHow
Create a boardClick New Board in the nav, enter a name, and click Create Board
Load a boardSelect a board from the Load an existing board dropdown and click Load Board
Switch boardsClick the current board name in the nav bar, then use Load Board

Tasks

ActionHow
Add a taskClick Add Item at the bottom of any column. Enter a title, optional description and priority, then click Add
Edit a taskClick on any card to open the edit dialog. You can update the title, priority, description, and comments
Move a taskDrag a card to a different column — the status is saved automatically
Set priorityChoose Low, Medium, or High in the Add or Edit dialog. A colour-coded icon appears on the card

Attachments

Files can be attached to any task and are stored in ArcGIS Online alongside the task data.

ActionHow
Upload a fileOpen a task → scroll to Attachments → choose file(s) → click Upload
Download a fileClick the filename link in the Attachments section
Delete a fileClick Delete next to the attachment

Export and copy

ActionHow
Export all tasksClick Export in the nav to download a CSV of all tasks across all columns
Copy a columnClick the icon in a column header to copy its tasks to the clipboard as tab-delimited text

Custom task ID prefix

By default task IDs are prefixed with T (e.g. T-42). To use a custom prefix, add prefix=XYZ to the layer's Copyright Text field in ArcGIS Online. For example, prefix=ABC produces IDs like ABC-T-1.

Dark mode

Click the / button in the top-right corner to toggle between light and dark mode. Your preference is saved and applied across all pages.
